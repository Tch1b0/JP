# JP

## Purpose

**JP** is the name of **my personal website** on which I am able to upload posts via http-requests.

## Setting Up

```
$ git clone https://github.com/Tch1b0/JP
```

```
$ cd ./JP/
```

```
$ npm install
```

```
$ npm run build
```

```
$ docker-compose up -d
```

## Requirements

-   Docker
-   npm
